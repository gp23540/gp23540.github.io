Work Samples

Here is some of my work from my Junior year of shop.
In particular, the applications I worked on
for Visual Basic assignments.


This app lets the user view any specific month's electric savings from setting up a smart home through a combo box and then click a button that shows more detailed statistics. The detailed statistics include the average monthly savings and which month had the most savings. This program was an important experience as I got to learn how to deal with arrays and reading text files (the amount saved came from a text file named savings.txt).


This app determines how many patients from today have a high cholesterol level and the patients' average cholesterol level. Those with a high cholesterol level are written to a file named consult.txt for further consulting. It also has a built-in option to view patients.txt, which contains every patient's name and their corresponding cholesterol level. Although similar to the program above, this program touched on writing to text files using an array's values, which the savings app did not need. Therefore, I can say that this program was also a significant experience.


This app displays the top 10 selling games and allows the user to select one of the game titles to display amount sold. A second form displays these games in alphabetical order.
